Supercast Extension Test Page

Testing Instructions:

This page is designed to help you test the Supercast Chrome extension. Press the keyboard shortcut Alt + Shift + S to toggle the overlay.

You can also click the button below to test the overlay manually:

Features to Test:

  • Toggle Overlay: Use the keyboard shortcut or button to show/hide the overlay
  • Minimal Mode: The overlay should initially show in minimal mode with just a command input and three buttons
  • Maximize: Click the maximize button to expand the overlay and show additional content
  • Pin Site: Test the pin button to add this test page to your pinned sites
  • Reading List: Test adding this page to your reading list
